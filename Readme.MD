A Streamlit-powered AI application for generating high-quality software documentation.

Features

8 Document Types: Proposal, BRD, SRS, TRD, HLD, LLD, API Docs, Test Plans.

Interview Mode: The AI acts as a Solution Architect to gather requirements via chat.

Mermaid.js Support: Automatically generates and renders architecture diagrams.

Export: Download documents in Markdown format.

Setup

Install Requirements:

pip install -r requirements.txt


Run the App:

streamlit run metamorphosis_app.py


API Key:

You will need a Google Gemini API Key.

Get one for free at Google AI Studio.

Enter it in the sidebar when the app launches.

Usage

Select your Document Type from the sidebar.

Chat with the AI Architect to provide project details.

Click Generate Final Document when finished.

Preview and Download your professional document.
